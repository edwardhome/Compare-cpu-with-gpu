# Compare-cpu-with-gpu
## GPU與CPU算力比較程式

## 需求套件
### tensorflow
### matplotlib

建議使用 **Anaconda** 環境

安裝 tensorflow 如下
``` conda install tensorflow ```

修改 matplotlib 
使用 pip 管理器
`pip uninsatll matplotlib` 移除原有的套件包

再使用
`pip install matplotlib` 安裝最新版本的資料包

`pip show matplotlib ` 確認版本為3.3.0

`python main.py`執行程式
